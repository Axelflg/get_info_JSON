PROGRAM NAME: 	Find_info.py					      
DESCRIPTION:  	Development of a data scraping program. Firstly loads the 
				'puntos_de_interes.JSON' file and get the name for each place. Afterward, 
				gets the time that a person spends in a place through Selenium and finally,
				gets the average time through regular expressions.
				
AUTHOR:     	Axel Flores Guarneros. 					      						      

PREREQUISITES:	Before to execute Find_info.py you should have the following file on 
				your folder:
					-> puntos_de_interes.json

				This file was generated by data_processing.py program. 

NOTES:		  	In addition, you need to install some dependencies to run it successfully: 
					-> requests
					-> re
					-> unittest
				Also, import some dependencies from selenium

The Find_info.py program updates the puntos_de_interes.json file:
	-> Add the 'nombre' property
	-> Add the 'tiempo_medio' property
